# 🚀 AI Skill Gap Analyzer

AI-powered FastAPI backend that analyzes employee skill gaps and recommends personalized learning paths.

---

## 📌 Project Overview

This project helps organizations identify the gap between current employee skills and required skills for specific roles. It provides intelligent recommendations to bridge those gaps through structured learning paths.

---

## ✨ Features

- Skill comparison engine  
- Gap detection logic  
- Learning path recommendation  
- REST API built with FastAPI  
- Automatic API documentation via Swagger UI  

---

## 🛠 Tech Stack

- Python  
- FastAPI  
- Uvicorn  
- Pydantic  

---

## 📂 How to Run Locally

1. Clone the repository:

git clone https://github.com/SrutiDevS/ai-skill-gap-analyser.git

2. Navigate into the folder:

cd ai-skill-gap-analyser

3. Install dependencies:

pip install -r requirements.txt

4. Run the server:

uvicorn main:app --reload

5. Open in browser:

http://127.0.0.1:8000/docs
